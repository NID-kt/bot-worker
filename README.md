## Run bot-worker

### Prerequisites

- Node.js v20.x
- corepack v0.20.0 or later
- `corepack enable pnpm`

### Install dependencies

```bash
pnpm install
```

### Production Build

```bash
pnpm build
pnpm start
```

### Development Build with Hot Reload

```bash
pnpm dev
```
